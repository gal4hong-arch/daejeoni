# 지자체 내부직원용 AI 업무지원 플랫폼 PRD
Version: 2.0 (통합 재설계)
Status: Final Draft

---

# 1. 제품 정의

## 1.1 한 줄 정의
사용자의 연속 대화를 이해하고, 내부 문서와 현행 법령을 기반으로  
업무 주제를 자동 분류하여 질의응답, 보고서 작성, 공문 작성, 검토, 시뮬레이션을 지원하는  
**주제 인식형 행정 AI 업무지원 플랫폼**

---

# 2. 핵심 컨셉

## 기존 방식
- 사용자가 세션을 직접 관리

## 변경 방식 (핵심)
- 사용자는 그냥 대화만 한다
- 시스템이 주제를 자동 분류한다
- 업무 단위(Topic Session)를 자동 생성한다

---

# 3. 사용자 경험 목표

사용자는:

- 세션을 신경 쓰지 않는다
- 그냥 질문하고 이어간다
- 시스템이 알아서 업무 단위로 정리한다
- 나중에 안건별로 결과를 확인한다

---

# 4. 핵심 기능

## 4.1 연속 대화 (Conversation Stream)
- 하나의 채팅 흐름
- 모든 메시지는 이 스트림에 저장됨

## 4.2 자동 주제 분류 (Topic Session)
시스템이 자동으로:
- 주제 추출
- 기존 주제 연결 or 신규 주제 생성
- 안건 제목 생성

## 4.3 문서 기반 질의응답
- 다중 문서 선택
- 근거 기반 답변
- 출처 표시

## 4.4 법령 실시간 조회
- 법제처 API 연동
- 현행 법령/조례 조회
- 답변에 자동 반영

## 4.5 문서 생성
- 보고서
- 공문 본문
- 설명자료
- 의회 답변자료

## 4.6 에이전트 검토
- 상급자 검토
- 법령 정합성 검토

## 4.7 시뮬레이션
- 의회 질의응답
- 상급자 리뷰

## 4.8 사용자 AI 설정
- API Key 등록
- 모델 선택
- 작업별 모델 설정

---

# 5. 자동 세션 관리

## 5.1 시스템 동작
각 메시지마다:
1. 주제 추출
2. 기존 topic과 비교
3. 연결 or 신규 생성

## 5.2 사용자 개입 최소화
사용자는 기본적으로:
- 세션 생성 안함
- 세션 선택 안함

## 5.3 보조 기능
- 안건 목록 보기
- 안건 병합/분리
- 제목 수정

---

# 6. 사용자 시나리오

## 시나리오 1
- 질문 → 답변 → 이어서 질문
→ 자동으로 같은 안건 유지

## 시나리오 2
- 완전히 다른 주제 질문
→ 자동으로 새로운 안건 생성

## 시나리오 3
- “보고서로 만들어줘”
→ 현재 안건 기준 문서 생성
